◆作品説明
コインを入手してスコアを稼ぎ、様々なギミックを乗り越えゴールを目指しましょう！
プレイヤー全員がゴールした時にスコアが一番大きいプレイヤーの勝利となります。

◆担当箇所
・ジャンプやパンチなどのモーション
・パッド対応
・メッシュ背景
・スコアの比較による順位付け
・ゴール後の動き
・複数プレイヤーの対応
・動く障害物の作成
・プレイヤーセレクト時の全員の完了処理
・カメラ関係の処理

◆チーム制作で心がけていたこと
・常に話し合いをして、チームの考えと自分の考えのズレがないか確認する
・わからないところがあれば遠慮せず周りに聞く
・自分の仕事が終わったらすぐに次の仕事に取り掛かるようにした
・自分の仕事の進捗具合などを常に報告する

◆工夫した点・苦労した点
・ただコインを取るだけではなく、ギミックによる妨害を重視した所。
・プレイヤー同士の掛け合いを増やすべく、ジャンプとパンチを追加した所。
・プレイヤー毎に個性を付ける為、色合いや顔のパーツをカスタマイズ出来る様にした所。
・ゲームのリトライ性を増やすべく、ステージを3種類 + ランダムを実装した所。
・当たり判定 <- 動くオブジェクト同士の挙動をどう誤魔化すか工夫した所
・ゴール処理 <- プレイヤーを指定の位置に動かし振り向かせて順位をソートするの一連の流れを実装する所
・タイトル名 <- モチーフは早めに決まったものの世界観等が定まらず、デザイン面が大きく停滞してしまった所
